#Sales Data Analysis

This is a Python project that analyzes sales data using Pandas and Matplotlib.

## Features
- Reads sales data from a CSV file
- Calculates total revenue
- Finds the best-selling product
- Shows sales by product
- Displays bar chart and pie chart visualizations

## Technologies Used
- Python
- Pandas
- Matplotlib

## Files in this Project
sales_analysis.py -> Main Python program  
sales_data.csv -> Dataset used for analysis

## How to Run
1. Install required libraries:
pip install pandas matplotlib

2. Run the program:
python sales_analysis.py

## Output
The program will:
- Print sales analysis in the console
- Display a bar chart of product revenue
- Display a pie chart of product sales distribution
